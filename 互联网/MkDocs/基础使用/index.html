    主要教程

    • mkdocs
    • mkdocs-material

    安装配置

    pip install mkdocs

    常规使用

    1. 创建工程：mkdocs new project_name
    2. 运行：mkdocs serve
    3. md文件的第一行留空，第二行输入---，这样mkdocs标题可显示为文件名

    实现中文搜索

    1. 安装mkdocs-materials：pip install mkdocs-material
    2. 在yml文件加上如下配置：
    theme:
  name: material
  navigation_depth: 10

plugins:
    - search:
        lang:
            - en
            - ja
        separator: '[\s\-\.]+'
